Tic Tac Toe

This project is a Tic Tac Toe game built using HTML, CSS, and JavaScript. It features an interactive 3x3 grid where two players can take turns marking their moves. The game logic, implemented in JavaScript, checks for winning combinations or a draw, providing a simple and engaging user experience.

Stone,Paper & Scissor

This project is a Stone Paper Scissors game created using HTML, CSS, and JavaScript. It allows two players (or a player vs. computer) to choose between stone, paper, or scissors. The JavaScript logic determines the winner based on the classic game rules: stone crushes scissors, scissors cuts paper, and paper covers stone. The interface is simple and interactive, providing instant results..
